# 💎 Luxury Wedding Animation Project

This is a cinematic AI-generated wedding animation project.

## 🎬 Concept
A royal Northern Nigerian luxury wedding film featuring:
- Elite indoor wedding hall
- Bride and groom on royal chairs
- Live musicians moving around stage
- Money spraying scenes
- Celebrity guests
- Romantic cinematic ending

## 🎯 Goal
To create a structured AI animation workflow using mobile tools only.

## 🧠 Style
Luxury, cinematic, slow motion, high-end wedding aesthetic.

## 🛠 Tools used
- AI image generation tools
- AI video generation tools (Runway / Pika / Kling)
- Mobile video editing (CapCut)
